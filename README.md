# 🤖 Avito → Telegram Notifications Bot

Бот для **личного аккаунта Avito**, который:

* 📩 получает **новые сообщения с Avito**
* 📦 показывает **название объявления**
* 👤 имя пользователя
* 🕒 время сообщения
* 🔁 позволяет **отвечать на Avito прямо из Telegram** (через reply)

Работает **локально**, без серверов, FastAPI и вебхуков.

---

## ✨ Возможности

* ✅ Автоматическое получение и обновление Avito access_token
* ✅ Уведомления только о **новых сообщениях**
* ✅ Игнор системных сообщений и ботов
* ✅ Ответы из Telegram → Avito
* ✅ Один файл `bot.py`
* ✅ Поддержка Windows 10 / 11

---

## 📦 Требования

* Python **3.9+** (рекомендуется 3.10–3.12)
* Аккаунт Avito с доступом к Avito API
* Telegram-бот (через @BotFather)

## 🔑 Настройка

Открой файл `bot.py` и заполни настройки:

### Telegram

```python
TELEGRAM_TOKEN = "TOKEN_ОТ_BOTFATHER"
TELEGRAM_CHAT_ID = 123456789  # твой chat_id
```

Как получить `chat_id`:

* напиши боту любое сообщение
* временно добавь `print(update.message.chat_id)` в обработчик
* или используй бота @userinfobot

---

### Avito API

```python
AVITO_CLIENT_ID = "client_id"
AVITO_CLIENT_SECRET = "client_secret"
```

Получаются в личном кабинете Avito API.

❗ Токен **получается автоматически**, вручную ничего обновлять не нужно.

## ▶️ Запуск бота

```cmd

```

Ожидаемый вывод:


После запуска:

1. Напиши боту `/start`
2. Получай новые сообщения Avito
3. Отвечай **reply** → сообщение уйдёт в Avito

---

## 🧠 Как это работает

* Бот каждые `CHECK_INTERVAL` секунд опрашивает Avito API
* Новые сообщения отправляются в Telegram
* ID сообщения Telegram связывается с чатом Avito
* Ответ через reply → сообщение отправляется обратно в Avito

---

## ⚠️ Известные особенности

* При перезапуске бота последние сообщения могут прийти повторно
  (данные хранятся в памяти)
* Avito API иногда возвращает историю, а не только unread

➡️ При необходимости можно добавить сохранение в файл
